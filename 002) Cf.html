웹 문서 만들기 

          웹 문서의 구성은 '!document html'로 시작하여 html,head,body 의 영역으로 구성되어 있다.
          1) document html  : 현재 문서가 작성된 언어가 'HTML' 이라는 표시
          2) html ~ /html   : 문서의 시작과 끝이며, 내용을 읽어 화면에 표시
            ?) 특정 언어로 정하는 이유 : 특정 언어 검색에 필요할때 사용
          3) head ~ /head   : 문서 해석 정보
            *) meta : 데이터에 관한 데이터
                - 웹 문서 관련 정보 지정시 사용
                    Ex) <meta name="keywords" content="html의 구조"> : 웹 문서의 키워드
                    Ex) &lmeta name="description" content="html의 설명"> : 웹 문서의 설명
                - 화면에 글자를 표시 할 때 어떤 인코딩을 사용할지 지정하는 것
                - 웹 서버 기본 : UTF-8(영어)
            *) title : 웹 페이지 이름
                    Ex) <title> 웹 페이지 이름 </title>
                - 웹페이지 이름 설정
          4) body ~ /body   : 화면에 나타내는 내용, => 표현할 모든 내용
